# Atividades — README

Este repositório contém pequenos exercícios em HTML/CSS e JavaScript usados para praticar conceitos básicos.

## Estrutura e descrições

- `atletas.js` - Script que contém uma lista de atletas e a classe `Jurados`. Calcula a média das notas de cada atleta e imprime no console uma avaliação com nome, notas e média.
  - Executar: `node "DEV Start/atletas.js"`
  - Link: [DEV Start/atletas.js](https://github.com/zwei-han-der/atividades/blob/main/DEV%20Start/atletas.js)

- `notas-atletas.js` - Define a classe `Atleta` com métodos para calcular categoria por idade, IMC (índice de massa corporal) e média das notas. Inclui um exemplo de instância que imprime informações do atleta no console.
  - Executar: `node "DEV Start/notas-atletas.js"`
  - Link: [DEV Start/notas-atletas.js](https://github.com/zwei-han-der/atividades/blob/main/DEV%20Start/notas-atletas.js)

- `login.html` - Página HTML simples de formulário de login (username e password) com estilos embutidos. Projetado para praticar HTML e CSS.
  - Abrir no navegador: clique duas vezes no arquivo ou arraste para o navegador.
  - Link: [Pessoal/login.html](https://github.com/zwei-han-der/atividades/blob/main/Pessoal/login.html)

- `praktika.html` - Página HTML com um componente de navegação (inspirado no design "Praktika") contendo botões SVG estilizados. Também para praticar HTML/CSS.
  - Abrir no navegador.
  - Link: [Pessoal/praktika.html](https://github.com/zwei-han-der/atividades/blob/main/Pessoal/praktika.html)

- `bhaskara.js` - Script que implementa a classe `Bhaskara` para calcular as raízes de uma equação quadrática usando a fórmula de Bhaskara. Calcula delta e verifica se existem raízes reais, imprimindo os resultados no console.
  - Executar: `node "Pessoal/bhaskara.js"`
  - Link: [Pessoal/bhaskara.js](https://github.com/zwei-han-der/atividades/blob/main/Pessoal/bhaskara.js)

- `estados.html` - Página que demonstra um formulário de seleção de Estados e Cidades. Contém um objeto `estados` com nomes e cidades para cada unidade federativa e carrega jQuery via CDN.
  - Abrir no navegador. Requer conexão com a internet para carregar jQuery do CDN.
  - Link: [Senac/estados.html](https://github.com/zwei-han-der/atividades/blob/main/Senac/estados.html)
